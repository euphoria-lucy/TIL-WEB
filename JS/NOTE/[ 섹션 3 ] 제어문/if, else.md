# 🍀 if / else

## 🧸 `if` 문

```javascript
// 한 줄 코드
if (조건) 문장;

// 여러 줄 코드 - 블록문 사용
if (조건) {
  문장;
}
```

<br>

## 🧸 `if ~ else` 문

```javascript
if (조건) {
  문장;
} else {
  문장;
}
```

### ⛄ 중첩 사용

```javascript
if (조건) {
  if (조건) {
    문장;
  } else {
    문장;
  }
} else {
  문장;
}
```

<br>

## 🧸 `if ~ else if ~` 문

```javascript
if (조건) {
  문장;
} else if (조건) {
  문장;
} else {
  문장;
}
```
