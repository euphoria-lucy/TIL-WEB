# 🍀 while과 do while

<br>

## 🧸 `while` - 조건이 참인 동안 반복수행

```javascript
while (조건) {
  문장;
}
```

<br>

### ⛄ 무한루프

```javascript
// 인위적인 무한반복에 널리 쓰이는 코드
while (true) {
  문장;
}
```

<br>

### ⛄ `break` 문으로 무한반복 탈출 가능

<br>

### ⛄ `continue` , `break` 사용

<br>

## 🧸 `do while` - **일단** 수행하고 조건을 평가

```javascript
do {
  문장;
} while (조건);
```
