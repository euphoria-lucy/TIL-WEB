# 🍀 클래스

<br>

## 🧸 `constructor` 메서드

- 인스턴스 생성 시 인자를 받아 프로퍼티를 초기화함
- 클래스에 **하나만** 있을 수 있음
- 다른 메서드 이름을 쓸 수 없음
- 기본값 사용 가능
- 필요없을 시 생략 가능

<br>

## 🧸 클래스의 메서드

<br>

### 💡 생성자 함수에 넣은 함수의 차이 - **프로토타입** 으로 들어감

<br>

## 🧸 필드 (field)

- `constructor` 밖에서, `this.~` 없이 인스턴스의 프로퍼티 정의
- Babel로 해결 가능

<br>

## 🧸 **정적** static 필드와 메서드

- 인스턴스의 수와 관계없이 메모리 한 곳만 차지
- 인스턴스 없이 클래스 차원에서 호출
- ⚠️ 정적 메서드에서는 정적 필드만 사용 가능
