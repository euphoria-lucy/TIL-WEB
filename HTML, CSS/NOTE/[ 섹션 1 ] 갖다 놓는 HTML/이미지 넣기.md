# 🍀 이미지 넣기

`<img src="(이미지 파일 경로)" alt="(대체 텍스트)" title="(툴팁 텍스트)">`

- alt : 사이트가 없어지거나 스크린리더로 듣는 사람들을 위해 작성하는 텍스트

- title : 이미지에 커서 올렸을 때 나타나는 텍스트

<br>

## 🧸 이미지 태그 속성

| 속성     | 설명          | 비고                                  |
| -------- | ------------- | ------------------------------------- |
| `src`    | 원본파일 경로 | 절대경로 또는 상대경로                |
| `alt`    | 대체 텍스트   | 스크린 리더, 원본파일 무효시          |
| `title`  | 툴팁          | `alt`의 대체제나 반복이 되어서는 안됨 |
| `width`  | 너비          | 픽셀 단위의 정수                      |
| `height` | 높이          | 픽셀 단위의 정수                      |
